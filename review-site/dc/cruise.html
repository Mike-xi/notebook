蛟人半潜器 · 自主巡航演示

这是什么

网页只是"数字水池"（物理仿真 · 传感器仿真 · 渲染）；巡航算法是同文件夹的 蛟人核心 Python 包（与装机运行同一份代码），每 10ms 由它给出桨 / 舵 / 压载指令。

三层框架

感知：JY901 + MS5837 + 艏部单目 → 视惯压融合定位、目标检测-跟踪；
决策：航点状态机 + ILOS 抗流制导 + 避障判定；
控制：M1 整定航向环 + 浮潜协同变深级联。

场景与规避（每段航路都被挡）

🐟 大鱼（潜深不够只能绕）：一条顶流悬停在航线正中、一条横穿河道、一条顺流漂游；
🌉 北桥墩：正挡在 2→3 航段上 → 绕行过桥；
🪨 礁石：正挡在 3→4 航段上 → 绕行；
🚤 渡船：机器人临近才离岸横穿 → 下潜穿越；
🟡 琥珀色立柱：可用鼠标拖动去挡路，考验临时重规划。

操作
