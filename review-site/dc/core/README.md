# 蛟人核心 —— 自主巡航算法（阶段一 + 阶段二 集成，部署级）

这是项目**唯一的一份**巡航算法：在仿真里验证的、将来装到半潜器上跑的，都是这套 `Lib\`。
组织方式沿用学森挑战营 `JiaoRen`（入口 + Lib 类库 + 多线程）。

```
蛟人核心\
├─ JiaoRen.py            实物入口（树莓派）：传感器流/视觉/控制环/控制台 四线程
├─ sim_entry.py          仿真接线层：引擎展示.html 与 可视化.html 都从这里调用
├─ 任务航点.json          （可选）装机任务航点，没有则用内置示例
└─ Lib\
   ├─ Params.py          全部参数与整定结论（M1 增益 / ILOS / 避障余量 / 执行器标定）
   ├─ Sensors.py         实物驱动：JY901(I2C) / MS5837 / 单目相机 → 统一弧度制字典
   ├─ StateEstimation.py 视惯压融合：姿态互补滤波 + 深度锚 + 航位推算/VO修正
   ├─ Vision.py          检测→α-β跟踪→短时记忆局部障碍地图（静态零速度先验）
   ├─ Guidance.py        航点状态机 + ILOS 制导 + 双通道规避 + 经济巡航
   ├─ Control.py         M1 航向环 + 浮潜协同变深级联（阶段一整定成果）
   └─ Actuators.py       双桨(setMotor) / 尾舵(1500±500µs) / 压载 输出，无硬件自动 Mock
```

## 数据流（实物与仿真完全一致）

```
JY901 / MS5837 / 相机                    100Hz 控制拍
     │                                        │
     ▼                                        ▼
StateEstimation（融合估计） ──► Guidance（20Hz 决策：ILOS+避障） ──► Control ──► Actuators
     ▲                              ▲
     └── VO/SLAM 定位（阶段二）      └── Vision（10Hz 跟踪，检测前端 detect() 挂载点）
```

## 三种跑法

1. **实物**：`python3 JiaoRen.py`（树莓派上；PC 上运行则自动干跑模式）
2. **综合场景仿真**：上级文件夹双击 `启动演示.bat` → 浏览器打开引擎展示，
   网页自动加载本文件夹的 Python（网页只做水池物理+传感器仿真+渲染）
3. **可视化对比**：可视化.html →「🐍 载入蛟人核心」选中本文件夹，
   与内置基准算法同场对比
