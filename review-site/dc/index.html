阶段成果 MILESTONES

阶段成果要点
阶段一 动力学建模与运动控制 5 自由度动力学建模与验证（RK2 数值水池）；M1 航向环参数整定 kp=3.0 / kd=1.5； 浮潜协同变深使调节时间缩短 29%。
阶段一 ILOS 抗流制导 积分视线法补偿横流，横流场对比中稳态横偏降低 68%；经济巡航油门伺服。
阶段二 多模态感知 视惯压融合定位（深度计尺度锚抑制视觉-惯性尺度发散）；水下图像增强对比实验； 目标检测 → α-β 跟踪 → 短时记忆局部障碍地图。
阶段三 图论一致性编队 分布式参考观测器（二阶一致性 + pinning）+ 邻居相对队形反馈，个体仅需邻居信息， 拓扑可切换、个体可失效；地速闭环治欠驱动稳态滞后；5 种队形在线变换。
部署 装机同款算法包 仿真验证与实物装机共用同一份 Lib/： 实物入口 JiaoRen.py（四线程）， 仿真接线 sim_entry.py / swarm_entry.py。

运行说明 NOTES

算法真实在线运行，非预录动画。 进入场景后浏览器将从本站加载 core/ 下的 Python 算法包并通过 Pyodide（WebAssembly CPython）执行——首次加载约需十余秒，请耐心等待； 网页与 Python 每拍交换「传感器读数 → 执行器指令」，算法不读取任何仿真真值。 建议使用桌面端 Chrome / Edge 浏览，场景内操作说明见各场景页面。
