环境说明

算法接入（两种方式）

① 实时仿真：点「载入 Python 算法」选 .py 文件（可多选），文件里按约定写 update(state, env, dt) 返回 (左桨,右桨,舵角,注水)，即出现在算法下拉框可切换对比， 模板见 仿真\算法示例_LOS.py；也可在控制台用 JIAOREN.registerController() 注册 JS 版。 点「🤖 载入蛟人核心」可直接载入本项目的正式算法包（蛟人核心\，JiaoRen 结构，装机同款）。
② 算法分析：回放 Python 正式仿真（jiaoren_sim）导出的 输出\*.json， 用于严谨的定量分析与报告出图。

虚拟传感器（与实物外设对应）

SLAM/感知传感器套件 = 相机 + IMU + 深度计（含噪声与采样率）。算法应只用传感器数据、不偷看真值，移植实物才无缝：
· read_imu() — JY901：欧拉角°、角速度°/s、比力m/s²，100Hz
· read_depth() — MS5837：深度m（σ3mm）、水压kPa，25Hz
· read_camera() — 艏部单目：160×120 灰度帧，≤20Hz（SLAM/视觉接入点；池壁设有特征标志板，可用 🏁 开关模拟纹理丰富/贫乏）
JS 侧对应 JIAOREN.sensors.*；点「📷 机载相机画面」可实时查看相机视角； 传感器版算法模板见 仿真\算法示例_视惯压融合.py。
