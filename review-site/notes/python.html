01

Python 概述

Overview · Setup · First Line

Python 是一门动态、解释型、面向对象的高级语言，由 Guido van Rossum 于 1989 年开始设计，1991 年发布首个版本。本课程采用 Python 3.10.10 + Jupyter Notebook / VS Code 环境。

语言核心特征

解释型 / 跨平台

源代码即可执行；通过解释器运行于 Windows / Linux / macOS。

动态类型

变量无需声明，类型由赋值时的值决定；运行时可重新指向不同类型对象。

语法精简

用缩进表达代码块，省去花括号；可读性极高。

丰富数据结构

内置 list、tuple、dict、set；标准库与第三方包生态庞大。

面向对象

一切皆对象；同时支持过程式、函数式风格。

健壮性

提供异常机制、垃圾回收、堆栈跟踪等保障。

开发环境

  • Jupyter Notebook：基于网页的交互式环境，单元格执行、即时输出，适合学习与数据分析（推荐通过 Anaconda 安装）。
  • VS Code：通用代码编辑器，安装 Python 扩展后即可获得调试、智能提示、终端集成。
  • 命令行：直接键入 python 进入交互模式，提示符 >>>。

第一行代码：print

print("Hello, Python world!")   # 输出字符串
print(2 + 3)                    # 输出表达式结果 5
# 注释以 # 开头；所有符号必须使用英文标点
关键习惯
所有标点必须是英文符号（括号、引号、冒号、逗号）。错误时 Python 会给出 traceback，提示行号与错误类型，应学会阅读这些信息。
02

变量与数据类型

Variables · Types · Assignment

变量是一个可被赋值的标签，指向内存中的某个对象。Python 采用基于值的内存管理：相同值默认共享同一内存位置。

命名规则

  • 首字符必须是字母或下划线，其余可含字母、数字、下划线。
  • 不能使用空格、连字符或其他特殊字符；区分大小写（tax ≠ Tax）。
  • 不能使用 Python 关键字作变量名（if、else、def、class、True、None 等）。
  • 不建议覆盖内置函数名（list、str、type……）。
  • 风格建议：简短而具有描述性，多单词用下划线分隔（student_name）。

简单数据类型

类型含义示例
int整数num = 1
float浮点数（含小数点）cost = 2.99
str字符串（单/双引号皆可）name = "Ada"
bool布尔值，首字母大写flag = True / False

用内置函数 type(x) 查看变量类型；用 id(x) 查看其内存地址。

赋值方式

# 普通赋值
message = "Hello"

# 多重赋值（同值）
a = b = c = 0

# 多元赋值（按顺序对应）
x, y, z = 1, 'two', 3.0

# 变量交换（Python 特色）
a, b = b, a

类型转换

函数作用示例
int(x)转为整数int('42') → 42
float(x)转为浮点float('3.14') → 3.14
str(x)转为字符串str(2024) → '2024'
hex(x) / oct(x)整数转十六/八进制字符串hex(255) → '0xff'

用户输入：input

age = input('请输入年龄: ')   # input 返回值永远是字符串
age = int(age)                # 参与数学运算前必须转换
易错点
通过 input() 获得的内容始终是字符串，看起来像数字也不能直接做加减。要么用 int()/float() 转换，要么留作字符串处理。
03

字符串

Strings · Slicing · Methods

字符串是一系列字符，属于序列类型，可被索引、切片、拼接。它不可变 —— 所有"修改"方法都返回新字符串。

索引与切片

正索引从 0 开始；负索引从 -1（最后一个）反向计数。切片语法 s[i:j:k] 表示从 i 到 j-1（左闭右开）、步长 k。

s = "Hello, Python!"
s[0]        # 'H'
s[-1]       # '!'
s[:5]       # 'Hello'    省略起始 = 0
s[7:]       # 'Python!'  省略终止 = 末尾
s[::2]      # 'Hlo yhn'  每隔两个
s[::-1]     # 反转字符串

大小写与空白处理

方法返回值
s.upper()全部转大写
s.lower()全部转小写
s.title()每个单词首字母大写
s.capitalize()仅第一个字符大写
s.swapcase()大小写互换
s.strip() / lstrip() / rstrip()去除两端 / 左端 / 右端空白

拼接、重复、转义

first + " " + last     # + 拼接，两侧必须都是字符串
'Python' * 3            # 重复 → 'PythonPythonPython'
str(2024) + ' year'    # 数字必须先转字符串

# 转义字符
"\t"   # 制表符
"\n"   # 换行符
"\\"   # 反斜杠本身

分割与查找

方法说明
s.split(sep)按 sep 切分，返回列表（默认按任意空白）
s.rsplit(sep)从右端开始切分
s.splitlines()按行切分
s.partition(sep)切成 (head, sep, tail) 三元组
s.replace(old, new)将 old 全部替换为 new
'a' in s / not in成员检查，返回 True/False
len(s) / min(s) / max(s)长度 / 最小字符 / 最大字符
04

列表 / 元组 / 字典

List · Tuple · Dict

Python 内置四大数据结构中最常用的三种：列表是有序、可变的元素序列；元组是有序、不可变的轻量集合；字典是键-值映射。

列表 List

用方括号创建，元素任意类型，可随时增删改。

structures = ['steel', 'concrete', 'timber', 'masonry']
structures[0]          # 'steel'
structures[-1]         # 'masonry'
structures[1:3]        # ['concrete', 'timber']
structures[2] = 'wood' # 修改元素
方法说明
lst.append(x)末尾追加单个元素
lst.insert(i, x)在索引 i 处插入元素，原元素后移
lst.extend(other)末尾追加另一列表的全部元素
del lst[i]按索引删除（不返回值）
lst.pop(i=-1)删除并返回索引 i 处元素（默认末尾）
lst.remove(x)删除第一个等于 x 的元素
lst.count(x)统计 x 出现次数
lst.index(x)返回 x 第一次出现的索引
lst.sort(reverse=False)原地排序，永久改变列表
sorted(lst)返回排序后的新列表，原列表不变
lst.reverse()原地反转
len(lst)元素个数

元组 Tuple

用圆括号创建，不可修改元素，常用于表达"一组应当被绑在一起、不应被更动"的值。

dimensions = (20, 5)
empty      = ()
single     = (20,)        # 单元素元组逗号不可省！
not_tuple  = (20)         # 这是整数，不是元组

# 元组本身不能修改，但可以重新绑定整个变量
dimensions = (200, 50, 1)   # 合法
dimensions[0] = 30           # 报错 TypeError

字典 Dict

用花括号创建，由键-值对组成；键必须不可变且唯一，值可以是任何对象。

grid = {'type': 'structured', 'points': 5000}

grid['type']                    # 访问 → 'structured'
grid['cells'] = 4               # 新增键值对
grid['points'] = 9              # 修改现有值
del grid['cells']                # 删除键值对
方法说明
d.keys()返回所有键的视图
d.values()返回所有值的视图
d.items()返回所有 (key, value) 二元组视图
d.get(key, default)取值；不存在返回 default 而非报错
d.pop(key)删除键并返回其值
d.popitem()弹出并返回一个键值对
d.update(other)用另一字典更新当前字典
d.setdefault(k, v)k 存在则取其值；否则插入并返回 v
d.clear()清空所有键值对

嵌套结构

列表与字典可以任意嵌套，是 Python 表示真实世界结构化数据的基础。

# 字典列表：管理一组同类对象
aliens = [
    {'color': 'green', 'points': 5},
    {'color': 'yellow', 'points': 10},
]

# 字典中嵌套列表：一个键对应多个值
pizza = {'crust': 'thick',
         'toppings': ['mushrooms', 'cheese']}

# 字典中嵌套字典：以键标识对象
users = {
    'aeinstein': {'first': 'Albert', 'location': 'Princeton'},
    'mcurie':    {'first': 'Marie',  'location': 'Paris'},
}
05

文件操作

File I/O · os Module

读写文件是数据处理的第一步。推荐使用 with open() as 结构，它会自动在合适的时候关闭文件，避免资源泄漏。

打开模式

模式含义
'r'只读（默认），文件不存在则报错
'w'写入；存在则清空原内容，不存在则新建
'a'追加；写在文件末尾，原内容保留
'rb' / 'wb' / 'ab'二进制模式（图片、音频等非文本）

两种打开方式

open + close（传统）
fo = open('foo.txt', 'w')
fo.write('hi')
fo.close()

必须手动 close；遇异常容易遗漏。

with open（推荐）
with open('foo.txt') as f:
    text = f.read()

代码块退出时自动关闭，更安全。

读与写

# 读取整个文件为一个字符串
with open('pi_digits.txt') as f:
    contents = f.read()

# 按行读取为列表
with open('pi_digits.txt') as f:
    lines = f.readlines()
    for line in lines:
        print(line.rstrip())     # readlines 会保留行末 \n

# 写入：write 不会自动换行，需要手动加 \n
with open('out.txt', 'w') as f:
    f.write('I love programming.\n')
    f.write('I love games.\n')

文件路径

  • 相对路径：相对于当前运行程序所在目录。Linux/macOS 用 /，Windows 用 \（建议在前面加 r 表示 raw 字符串以避免转义问题）。
  • 绝对路径：完整路径，与当前目录无关。例 r'C:\Users\me\data\x.txt'。

os 模块：文件夹与文件管理

import os

os.getcwd()                       # 当前工作目录
os.mkdir('myDir')                # 创建文件夹
os.rmdir('myDir')                # 删除空文件夹
os.rename('old.txt', 'new.txt')  # 重命名
os.remove('new.txt')              # 删除文件

运算符与表达式

Operators · Expressions

Python 提供算术、赋值、比较、逻辑、按位五类运算符，与运算对象组合即成表达式。

算术运算符

运算符含义示例
+ - * /加减乘除14 / 7 → 2.0（始终浮点）
**幂运算7 ** 2 → 49
//整除（向下取整）15 // 4 → 3
%取余15 % 7 → 1

赋值与自反赋值

x = 5            # 基本赋值
x += 3           # 等价于 x = x + 3
# 同理：-= *= /= //= **= %=

比较运算符

返回 True 或 False，可作用于数字、字符串（按字符 ASCII 值逐字比较）。

运算符含义运算符含义
< <=小于 / 小于等于> >=大于 / 大于等于
==相等!=不等
极易混淆
= 是赋值；== 是比较。把 if x = 5 写成赋值会引发 SyntaxError。

逻辑运算符

运算符规则
and两个操作数都为 True 才为 True（短路求值：左侧 False 即返回）
or有任一为 True 即为 True（左侧 True 即返回）
not单目运算符；取反
 
num = 15
(num % 2 == 0) and (num % 3 == 0)   # False（不能被2整除）
(num % 2 == 0) or (num % 3 == 0)    # True（能被3整除）

按位运算符（了解）

作用于整数的二进制位。常见操作：

运算符说明
&按位与：对应位都为 1 才为 1
|按位或：任一位为 1 就为 1
^按位异或：不同则为 1
~按位取反（单目）
<<左移：高位舍弃、低位补 0，等于乘以 2 的 n 次幂
>>右移：低位舍弃、高位补 0，等于整除 2 的 n 次幂
07

选择结构 if

Conditional Branching

通过判断条件决定执行哪一路代码。Python 用缩进而非花括号来表达代码块，缩进务必使用统一的 Tab 或空格。

三种形式

# 单分支
if num < 0:
    num = -num

# 双分支
if a > b:
    print('a 大')
else:
    print('b 大或相等')

# 多分支
if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'
elif score >= 60:
    grade = 'D'
else:
    grade = 'Fail'

嵌套与组合

条件可用 and、or、not 组合，避免过度嵌套。下方两种写法等价：

嵌套
if x > 0:
    if y > 0:
        print('第一象限')
逻辑组合
if x > 0 and y > 0:
    print('第一象限')
缩进守则
每个 if、elif、else 后必须有冒号，下一行必须缩进。同一分支内所有语句缩进相同；退出分支时取消缩进。
08

循环结构 for / while

Loops · break · continue

两种循环：for 适合已知次数或对序列迭代；while 适合条件控制的不定次循环。

for-in 循环

# 直接遍历序列
for ch in 'Python':
    print(ch)

# 用 range() 控制次数
for i in range(10):           # 0,1,...,9
    print(i)

for i in range(1, 100, 2):     # 1,3,5,...,99（奇数）
    print(i)

# for-else：循环完整执行后才执行 else
for x in seq:
    process(x)
else:
    print('循环正常结束')

range(start, stop, step)

调用产生序列
range(6)0, 1, 2, 3, 4, 5
range(4, 8)4, 5, 6, 7
range(4, 10, 2)4, 6, 8
range(10, 0, -1)10, 9, 8, …, 1（倒序）

stop 不可省略，且不包含在结果中（左闭右开）。

while 循环

num = 1
while num <= 50:
    if num % 3 == 0:
        print(num)
    num += 1           # 必须显式修改循环变量，否则死循环
while 关键
务必让循环条件能朝着"为假"的方向演进，否则会陷入死循环。两件事不能忘：① 循环前初始化控制变量；② 循环体内更新控制变量。

break 与 continue

break

立即跳出所在层的循环，不再判断后续条件。多重循环中只跳出一层。

for x in range(10):
    if x == 5:
        break     # 提前结束
    print(x)
continue

跳过本次循环剩余语句，直接回到循环头判断条件。

for x in range(10):
    if x % 2 == 0:
        continue  # 跳过偶数
    print(x)

循环嵌套

外层每执行一次，内层完整运行一遍。可用于二维表格、求最大公约数等问题。

# 求两正整数的最大公约数（辗转相减法）
while x1 != x2:
    if x1 > x2:
        x1 -= x2
    else:
        x2 -= x1
print(x1)            # 此时 x1 == x2 即为 gcd

函数基础

Functions · Definition · Return

函数把一段可复用的逻辑封装为一个名字。Python 的函数分为三类：内置函数（直接可用）、标准库函数（需 import）、自定义函数（用 def 创建）。

定义与调用

def max3(x1, x2, x3):
    """返回三个数中的最大值"""
    temp = x1 if x1 > x2 else x2
    return temp if temp > x3 else x3

result = max3(3, 7, 5)        # 调用，传入实参
  • def 关键字必须 + 空格 + 函数名 + 圆括号 + 冒号。
  • 函数体整体缩进；可选的首行文档字符串（docstring）描述功能。
  • return 必须在函数体内、可省略；省略则返回 None。

返回值的多种形态

# 返回单值
def sq(n): return n * n

# 返回多值（实为元组）
def stats(a, b): return a + b, a * b, a / b
s, p, q = stats(10, 3)         # 元组解包

# 返回列表
def stats(a, b): return [a+b, a*b, a/b]

# 返回字典
def person(first, last):
    return {'first': first, 'last': last}
10

参数传递

Positional · Default · Keyword · *args / **kwargs

Python 提供多种灵活的参数传递方式，可以同时混用，但要注意顺序与规则。

四种基本方式

类型说明
位置参数按顺序对应；顺序错就传错
默认参数定义时给出默认值，调用时可省略；必须放在形参表最右端
关键字参数调用时用 name=value 指定；顺序无关紧要
混合调用位置参数必须在关键字参数之前
 
def per(name, gender, city='Shanghai'):
    return [name, gender, city]

per('Tom', 'M')                       # city 用默认值
per('Tom', 'M', 'Beijing')              # 位置参数
per(name='Tom', gender='M')             # 关键字参数
per(gender='M', name='Tom')             # 顺序无关

不定长参数 *args 与 **kwargs

当无法预先知道参数个数时使用："收纳"多余的实参；用在调用时则是"拆分"。

*args：元组收纳
def count(name, *scores):
    total = sum(scores)
    print(name, total)

count('Tom', 76, 68, 99, 84)
# scores → (76, 68, 99, 84)
**kwargs：字典收纳
def stu(term, **persons):
    for k, v in persons.items():
        print(k, v)

stu('2024', Mary=90, Eric=75)

拆分调用

nums = [78, 99, 65]
scores = {'a': 90, 'b': 80}

count('Tom', *nums)         # 拆分列表 → count('Tom', 78, 99, 65)
stu('2024', **scores)      # 拆分字典 → stu('2024', a=90, b=80)

可变 vs 不可变参数

关键概念
Python 中函数对不可变对象（数、字符串、元组）的修改不会影响实参；对可变对象（列表、字典）的修改会直接反映到实参，因为它们共享同一内存地址。

作用域与模块

Scope · global · Modules · Standard Library

变量起作用的范围称为作用域。函数内部、外部、内置三种层级；模块则是把函数 / 类装入文件以便复用。

三种作用域

作用域说明
局部 Local函数 / 程序块内部创建的变量；函数返回即消亡
全局 Global文件层面定义的变量；可被各函数引用

栈与队列

Stack · Queue · Linear Structures

线性结构是有序的数据集合，每个数据项有唯一的前驱与后继。栈与队列是最常用的两种受限线性结构，区别在于增删元素的"端"。

栈 Stack — LIFO

"后进先出"（Last In First Out）。所有增删都发生在同一端 — 栈顶。典型应用：浏览器返回、撤销操作、括号匹配、进制转换。

抽象数据类型 Stack 的操作

操作语义
Stack()创建空栈
push(item)压入栈顶
pop()弹出并返回栈顶元素
peek()查看栈顶元素但不移除
isEmpty()是否为空
size()元素个数

用 list 实现 Stack

class Stack:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)        # 列表末尾作栈顶 → O(1)

    def pop(self):
        return self.items.pop()

    def peek(self):
        return self.items[-1]

    def isEmpty(self):
        return self.items == []

    def size(self):
        return len(self.items)
实现取舍
若把列表的首端当作栈顶，push/pop 复杂度变为 O(n)（因要移动所有元素）。用末端作栈顶则为 O(1)。

应用一：括号匹配

从左到右扫描，遇左括号入栈、遇右括号则与栈顶配对。栈最终为空 → 匹配成功。可扩展到 () [] {} 多种括号混用。

应用二：十进制转任意进制

核心算法：不断 n // base、n % base，余数顺序与输出顺序相反 —— 正好利用栈反转。

def to_base(n, base):
    digits = "0123456789ABCDEF"
    stack = Stack()
    while n > 0:
        stack.push(n % base)
        n = n // base
    out = ""
    while not stack.isEmpty():
        out += digits[stack.pop()]
    return out

队列 Queue — FIFO

"先进先出"（First In First Out）。一端入队（队尾 rear），另一端出队（队首 front）。日常排队、打印任务调度、进程调度、键盘缓冲都是队列。

操作定义

操作语义

enqueue(item)加入队尾
dequeue()从队首移除并返回
isEmpty()是否为空

class Queue:
    def __init__(self):
        self.items = []
    def enqueue(self, item):
        self.items.insert(0, item)     # 首端入队 O(n)
    def dequeue(self):
        return self.items.pop()           # 末端出队 O(1)
    def isEmpty(self):
        return self.items == []
    def size(self):
        return len(self.items)

经典应用：约瑟夫问题（热土豆）

n 人围圈传球，传 k 次后手中拿球的人出列；重复直到只剩一人。用队列模拟：队首即"当前持球者"，传球 = 出队再入队，传完 k 次后出队首并丢弃。

递归

Recursion · Divide & Conquer · Fractals

递归是将问题分解为规模更小的相同问题的解题策略；在程序中表现为函数调用自身。许多复杂问题用递归表述既简洁又优雅。

递归三定律

① 基本结束条件

最小规模的问题必须能直接、不再递归地解决，否则会无限调用导致 RecursionError。

② 减小规模

每次递归调用要朝着"基本结束条件"前进 —— 通常是缩短列表、减少整数、缩小区间等。

③ 调用自身

函数体内调用自身，传入"更小规模"的参数；用其返回值合成当前问题的解。

入门示例：数列求和

def list_sum(nums):
    if len(nums) == 1:            # ① 基本结束条件
        return nums[0]
    return nums[0] + list_sum(nums[1:])  # ② 缩小 + ③ 调用自身

整数转任意进制（递归版）

def to_str(n, base):
    digits = "0123456789ABCDEF"
    if n < base:                       # 最小规模
        return digits[n]
    return to_str(n // base, base) + digits[n % base]

递归调用的实现：调用栈

每次函数调用，系统把"现场"打包成栈帧压入调用栈；返回时弹出。递归层数过多会导致栈溢出。

import sys
sys.getrecursionlimit()        # 默认上限通常 1000
sys.setrecursionlimit(3000)     # 可调高，但要小心

经典递归问题

汉诺塔 Hanoi

将 n 个盘子从开始柱借助中间柱搬到目标柱，规则是大盘不能压小盘，每次只能搬一个。

def hanoi(n, start, mid, end):
    if n == 1:
        print(f'{start} → {end}')
        return
    hanoi(n - 1, start, end, mid)   # 顶上 n-1 盘 移到中间柱
    print(f'{start} → {end}')        # 最大盘 移到目标柱
    hanoi(n - 1, mid, start, end)   # 中间柱上 n-1 盘 移到目标柱

移动次数为 2ⁿ − 1。64 个盘子需 5849 亿年（每秒一次）。

迷宫探索

用二维列表表示迷宫（+ 表墙、空格表通道）。从某点出发，分别尝试向北、南、西、东走，每个方向再递归调用自身。关键：撒"面包屑"避免回头走入死循环。

基本结束条件：① 撞墙；② 踩到面包屑；③ 到达边缘（找到出口）；④ 四向都失败。

查找与排序

Search · Sort · Complexity

查找与排序是算法学习的起点。掌握不同方法的时间复杂度，才能在实际场景中作出合适选择。

查找算法

顺序查找 Sequential Search

从头到尾逐个比较，直到找到或遍历结束。无需数据有序。

def seq_search(lst, item):
    for i, x in enumerate(lst):
        if x == item:
            return True
    return False
情形最好最坏平均
数据项存在1nn/2
数据项不存在n（无序） / 提前结束（有序）nn / n/2

整体时间复杂度 O(n)。有序表能在不存在时提前退出，但量级不变。

二分查找 Binary Search O(log n)

仅适用于有序表。每次比较中间项，将待查范围缩小一半。

def binary_search(lst, item):
    if len(lst) == 0:
        return False
    mid = len(lst) // 2
    if lst[mid] == item:
        return True
    if item < lst[mid]:
        return binary_search(lst[:mid], item)
    return binary_search(lst[mid+1:], item)
分而治之
递归本质上就是分治策略，二分查找是其经典应用：把问题切成两半、解决其中一半（另一半丢弃）。但要注意切片操作本身有 O(k) 开销，工程实现常用左右索引代替切片。

冒泡排序

每趟相邻两两比较交换，让本趟最大者"冒泡"到末端。共 n − 1 趟。

def bubble(lst):
    for i in range(len(lst) - 1, 0, -1):
        for j in range(i):
            if lst[j] > lst[j+1]:
                lst[j], lst[j+1] = lst[j+1], lst[j]

选择排序

冒泡的改进：每趟只记录最大值的位置，最后再做一次交换。比对次数同冒泡，交换次数减为 O(n)。

插入排序

维护一个已排序子列表，每次把新元素插入到合适位置（移动而非交换，更省时）。在数据"接近有序"时表现极佳。

谢尔排序

插入排序的变体：先按较大间隔分组、组内插入排序，逐步缩小间隔到 1。利用插入排序在"接近有序"时的高效，整体快于 O(n²)。

典型的分治策略：将列表对半切分 → 递归排序两半 → 合并两个有序列表。

分裂 O(log n)，每层归并 O(n)，整体 O(n log n)。代价：需要 O(n) 额外空间。

快速排序 Quick Sort

选一个"中值"作枢轴（pivot），将列表分为小于和大于两半，分别递归排序。无需额外空间，但若枢轴选取不佳（如已有序列表选第一个），会退化为 O(n²)。改进：三点取样（取首、中、尾的中位数）。

如何选择
实际工程中并不是越快越好：① 若数据动态变化、查找少 → 顺序查找 + 无序表可能更经济；② 若空间受限 → 选择 / 插入排序原地操作；③ 一次性大数据排序 → 优先归并 / 快速排序；④ Python 内置 sort() 和 sorted() 使用 Timsort（归并 + 插入的混合），通常无需自己实现。

result = [

核心公式：求导 (xⁿ)′ = n·xⁿ⁻¹；积分 ∫k dx = kx + c、∫(1/x)dx = ln x + c；分部积分 ∫uv′dx = uv − ∫u′v dx。Python 计算定积分两条路：

方法一：解析解

先手算出原函数，再编写函数代入上下限。精确但复杂函数难求。

方法二：矩形法则

区间分成 n 份，Σ f(xᵢ)·dx 近似面积。n 越大越准，是数值积分。

def integrate_rectangles(func, a, b, n):
    dx = (b - a) / n
    return sum(func(a + i*dx) * dx for i in range(n))
integrate_rectangles(lambda x: x**2, 0, 1, 1000)  # ≈ 0.333

经典问题：斐波那契（递归 vs 迭代）

同一问题、两种写法，复杂度天差地别 —— 这是本章最重要的考点。

递归 O(2ⁿ)
def fib(n):
    if n <= 1: return n
    return fib(n-1) + fib(n-2)

树结构，大量重复计算。

迭代 O(n)
def fib(n):
    a, b = 0, 1
    for _ in range(n-1):
        a, b = b, a+b
    return b

环结构，逐步更新状态。

第 n 项递归用时迭代用时
300.272 s~0 s
4031.795 s~0 s

同一台机器数据，结论恒成立：迭代远快于递归。递归=自相似的"递推+回归"（A 调用 A）；迭代=用上次结果作下次输入（A 重复调用 B）。

import random, math
crossings = 0
for _ in range(n):
    center = random.uniform(0, spacing/2)
    angle  = random.uniform(0, math.pi/2)
    if center < (length/2) * math.sin(angle):
        crossings += 1
pi ≈ 2 / (crossings / n)
